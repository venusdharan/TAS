html2.WebView Sample

This is a super simple launch page for the html2.WebView sample.


Normal web pages and real web sites work just as expected:

  • wxPython Home
  • wxWidgets Home
  • Microsoft Home
  • Google Home


Resources can also be loaded from custom file systems:

  • wx.MemoryFSHandler: Pages can be loaded from an in-memory filesystem
  • wx.ArchiveFSHandler: Or from a ZIP file
  • wx.html2.WebViewHandler: A custom handler can also be implemented
